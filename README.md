# AutoMind Agent

AutoMind is a Markdown-based autonomous DevOps AI agent designed to be runnable by AI CLIs like Claude Code or Kilo Code.

## Project Rules
- **Markdown Only**: This project strictly uses `.md` files for agent logic, configuration, and state.
- **No Scripts**: Creation or execution of `.js`, `.py`, or `.sh` files by the agent is forbidden.
- **Documentation**: All errors and issues must be recorded in Markdown format within the `memory/` directory.

## Folder Structure

- `agents/`: Contains the core agent definitions.
- `memory/`: Contains configuration and state tracking for the agent.
- `README.md`: This file, providing an overview of the system.

## Responsibility
This file serves as the entry point and primary documentation for the AutoMind agent, explaining how to set up and interact with the system.
